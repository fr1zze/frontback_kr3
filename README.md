# 📘 Контрольная работа 3

## PWA приложение "Заметки"

---

## 📌 Описание проекта

В рамках 3 контрольной работы было разработано **прогрессивное веб-приложение (PWA)**
для создания и управления заметками.

Приложение поддерживает:

* работу **без интернета**
* установку как отдельное приложение
* push-уведомления
* взаимодействие с сервером

---

## 🚀 Функциональность

### 📝 Работа с заметками

* ➕ Добавление заметок
* ✏️ Редактирование
* 🗑️ Удаление
* 🧹 Очистка всех заметок

### 🌐 PWA возможности

* 📦 Установка приложения (Install App)
* ⚡ Быстрая загрузка
* 📡 Работа офлайн (Service Worker + Cache API)
* 🎨 Адаптивный интерфейс

### 🔔 Уведомления (Практика 17)

* Push-уведомления
* Отложенные напоминания
* WebSocket соединение с сервером

## ⚙️ Установка и запуск

### 1. Установка зависимостей

```bash
npm install
```

### 2. Настройка HTTPS (обязательно для PWA)

```bash
mkcert -install
mkcert localhost 127.0.0.1 ::1
```

### 3. Запуск сервера

```bash
npm start
```

### 4. Открыть в браузере

```
https://localhost:3001
```

---

## 📲 Установка приложения

После запуска:

1. Открыть сайт в Chrome
2. В адресной строке появится кнопка **"Установить приложение"**
3. Нажать → приложение установится как отдельное

## 🧪 Проверка работоспособности

### ✔️ Проверка офлайн режима

1. Открыть DevTools → Network
2. Включить **Offline**
3. Перезагрузить страницу
   👉 Приложение должно работать

---

### ✔️ Проверка уведомлений

* Разрешить уведомления в браузере
* Отправить уведомление с сервера
  👉 Уведомление должно появиться
